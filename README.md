# School-Canteen-Preorder-System

## 📖 Introduction

In educational institutions, students often face long queues and waiting times at canteen counters during breaks and lunch hours. This leads to inconvenience, wasted time, and overcrowding. To address this problem, **CanteenGo** provides a digital pre-ordering platform that allows students to browse available food items, place orders in advance, select collection times, and collect their meals efficiently.

The system is designed as a user-friendly web application that streamlines the food ordering process while providing canteen administrators with tools to manage orders, monitor sales, and improve overall service quality.

---

## 🎯 Aim of the Project

The primary aim of the project is to develop a web-based canteen pre-order system that enables students to order food online before arriving at the canteen, thereby reducing waiting times and improving service efficiency.

### Objectives

* To provide a simple and interactive food ordering platform.
* To reduce congestion and queues at canteen counters.
* To enable students to pre-book meals and collection slots.
* To provide real-time order management for canteen staff.
* To maintain order history and customer feedback.
* To improve the overall canteen experience through digital automation.

---

## 🛠️ Technologies Used

### Front-End Technologies

#### HTML5

* Used to create the structure and layout of the web application.
* Defines pages, forms, buttons, navigation menus, and content sections.

#### CSS3

* Used for styling and designing the user interface.
* Provides responsive layouts, animations, color themes, and modern visual effects.

#### JavaScript (ES6)

* Handles application logic and user interactions.
* Manages cart operations, order placement, filtering, notifications, feedback, and admin functionalities.

### Browser Storage

#### LocalStorage API

* Stores cart data, orders, feedback, and application settings.
* Enables data persistence without requiring a backend database.

### Development Concepts Used

* Single Page Application (SPA)
* Responsive Web Design
* DOM Manipulation
* Event Handling
* Form Validation
* Client-Side Data Storage
* Dynamic Content Rendering

---

## 📌 Project Overview

CanteenGo consists of multiple integrated modules including menu management, cart management, order processing, feedback collection, notification handling, and an administrative dashboard. Students can browse food items, place orders, and track their status, while administrators can monitor sales, update order statuses, and export order records.

The entire application runs within the browser and demonstrates the practical implementation of Front-End Web Development concepts using HTML, CSS, JavaScript, and LocalStorage.
